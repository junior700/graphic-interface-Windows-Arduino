Unzip the program folder
Open the program folder
Click on the executable file with the right mouse button
Access the "create desktop shortcut"
Go to the desktop
Run the program by clicking on the created shortcut
A login window will open

Click "OK"
A window will open for connection to the device.
Connect the Arduino through the USB port
Press the "Connection" button
The system will look for the Arduino port
As soon as you find it, the "COMx" port will appear in the window (x is the port number)
After the system completes the connection (1 second), it will open the main program window
In the main window,
enter the "Execute" menu
Appear 4 intens, click on "Dimmer" to explore the operation.
The dimmer has 4 channels, each from 0 to 100%. There are 8 dimmer addresses,
A, B, C, D, E, F, G, H. By default, comes program "A", but to change
using the arrow.
The "time" control with slide button, serves to change the transition time
between the current value and the new dimerization value sent. From 0 to 100 s.
Comes programmed with 0 seconds.
Close the window by clicking "OK".
Returns to the main window.
Choose "ON-OFF", to turn on and off lamps,
and "sensors", which shows the current value of sensors connected in the automation.
If the system disconnects during use,
use the "Connection" menu to reconnect.
To exit, enter the menu and "Exit".
